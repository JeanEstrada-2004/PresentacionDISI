# Presentación Web DISI - NANU TECH

Presentación tipo diapositivas construida solo con HTML, CSS y JavaScript.

## Uso

Abre `index.html` en el navegador.

Controles disponibles:

- Flechas derecha/abajo: siguiente diapositiva.
- Flechas izquierda/arriba: diapositiva anterior.
- `Home`: primera diapositiva.
- `End`: última diapositiva.
- `N`: mostrar u ocultar notas del expositor.

La última diapositiva visitada se guarda en el navegador para continuar desde el mismo punto.
