# Presentación Final DISI - NANU TECH

Presentación web tipo diapositivas para la defensa final del proyecto:

**Diseño e implementación de una plataforma híbrida web-móvil para la gestión operativa de flota en NANU TECH**.

La versión actual contiene 28 diapositivas y cubre problema, objetivos, justificación, Lean Canvas, alcance MVP, dimensionamiento, actores, SDLC, metodología, requerimientos, reglas de negocio, trazabilidad, UX, arquitectura, offline/sync, datos, implementación, pruebas, despliegue, operación, mantenimiento y cierre.

## Uso

Abre `index.html` en el navegador.

Controles disponibles:

- Flechas derecha/abajo o espacio: siguiente diapositiva.
- Flechas izquierda/arriba: diapositiva anterior.
- `Home`: primera diapositiva.
- `End`: última diapositiva.
