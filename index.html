Necesidad que justifica la solución

Problema operativo identificado

Se explicará cómo la baja trazabilidad de jornadas afecta el seguimiento operativo y retrasa la validación administrativa.

  • Registros dispersos entre canales manuales y digitales.
  • Dificultad para conocer el estado real de cada jornada.
  • Validación tardía de inicio, cierre y observaciones.
  • Menor visibilidad para decisiones operativas.

Trazabilidad baja

El recorrido de la jornada no queda unificado para consulta y auditoría.

Seguimiento lento

La información llega después del evento operativo relevante.

Evidencia fragmentada

El informe conectará fuentes, actores y reglas de negocio.

Núcleo funcional evaluable

Idea TIC seleccionada

El sistema se plantea como una solución web-móvil integrada: el administrador gestiona y consulta desde la web, el chofer registra eventos desde el canal móvil y el backend garantiza reglas comunes, persistencia y trazabilidad.

  • Web: administración, consulta, seguimiento y validación.
  • Móvil: operación del chofer, inicio, cierre y observaciones.
  • Backend: reglas, persistencia, seguridad y trazabilidad.
  • Arquitectura integrada: web para gestión, móvil para operación y backend para lógica, datos y trazabilidad.
